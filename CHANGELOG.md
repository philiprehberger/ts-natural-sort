# Changelog

## 0.2.0

- Add `naturalSortBy(array, key)` strongly-typed shorthand for sorting objects by a string key
- `naturalCompare` now recognizes leading negative integers (`-10` sorts before `-2`)

## 0.1.3

- Standardize README to 3-badge format with emoji Support section
- Update CI actions to v5 for Node.js 24 compatibility
- Add GitHub issue templates, dependabot config, and PR template

## 0.1.2

- Standardize package metadata

## 0.1.1

- Standardize README badges and CHANGELOG formatting

## 0.1.0

- `naturalSort()` for sorting arrays with natural ordering
- `naturalCompare()` for pairwise natural string comparison
- Case-insensitive and descending options
- Accessor support for sorting objects by a string property
